# 📸 Screenshots

Add your Power BI dashboard screenshots here.

## Recommended Screenshots to Include

| File Name | What to Capture |
|---|---|
| `01_overview_page.png` | Main KPI dashboard — Revenue, Margin, Units cards + top charts |
| `02_category_analysis.png` | Category revenue & profit margin breakdown |
| `03_employee_performance.png` | Employee leaderboard / bar chart |
| `04_regional_comparison.png` | West vs. North side-by-side |
| `05_discount_analysis.png` | Discount rate matrix or scatter |
| `06_data_model.png` | Power BI model view showing table relationships |

## How to Take Screenshots in Power BI

1. Open your `.pbix` file in Power BI Desktop
2. Navigate to the page you want to capture
3. Press `Windows + Shift + S` (Windows Snipping Tool) or use `Snagit`
4. Save as `.png` in this folder
5. Update the README to reference your screenshots using:

```markdown
![Dashboard Overview](./screenshots/01_overview_page.png)
```

> 💡 **Tip:** Screenshots are the #1 thing recruiters look at in a BI portfolio repo. Add at least 3.
